Recipes blog
Store all your favorite recipes in one place with this React app. The content is uploaded via Contentful (Headless CMS) and displayed on the website.

https://recipes-blog.netlify.app/

Installation instructions
Clone down the project to you computer:
$ git clone https://github.com/emeliepetersson/React-Recipes-blog
In the project directory, you can run:
$ npm start
This runs the app in the development mode. Open http://localhost:3000 to view it in the browser.
